# Ambang Flores

> A narrative-driven visual novel set in Flores, Indonesia.

## 📖 Overview

**Ambang Flores** is a visual novel that explores environmental and social issues through an interactive storytelling experience. Set against the backdrop of Flores, Indonesia, the game invites players to follow the narrative, make decisions, and witness the consequences of those choices.

The project was developed as an independent game with a strong focus on narrative, atmosphere, and local context.

## ✨ Features

* 📚 Story-driven visual novel experience
* 🌋 Setting inspired by Flores, Indonesia
* 🎭 Choice-based narrative progression
* 🌱 Themes related to environmental and social issues
* 💻 Developed using **Ren'Py**
* 🪟 Playable on **Windows**
